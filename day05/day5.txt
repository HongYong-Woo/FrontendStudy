2. 설정
3. 패턴

웹 브라우저는 '미디어 쿼리' 기능을 지원
미디어 쿼리란 미디어 타입에 따라 CSS를 다르게 적용하는 기능
프린터, 데스크탑, 스마트폰, 태블릿, 키오스크, 시작장애용 웹 브라우저 마다 적합한 웹페이지를 구현 할 수 있게 지원한다.

결론 : 다양한 장치에 따라 해상도 크기 및 비율이 제각각 다른 경우에 
미디어 쿼리를 적용한 웹 페이지는 개발 효율성과 유지보수애 큰 장점이 있다.
반응형 웹 페이지는 미디어 쿼리를 사용하여 개발한다.

2. 설정
    meta 태그 : 웹 페이지에 추가 정보를 제공
    <meta name="반응형 쇼핑몰 구현" content="신세계InC 자바개발자 미니프로젝트">
    <meta name="description" content="신세계InC 프로젝트 포트폴리오"> --검색엔진 노출

    2-2 viewport
    <meta name="viewport" content="width height initial-scale = 2.0 (초기확대 비율") > --초기화면 비율을 설정

    2-3 미디어 쿼리
        @media (<미디어 쿼리>) {
            CSS 코드
        }

        <link rel="stylesheet" href="파일이름" media="미디어쿼리"> 적용
